# @ciph/devtools-client

Development-only floating panel showing plain request/response data.

## Setup

```tsx
import { CiphDevtools } from "@ciph/devtools-client"

export function App() {
  return (
    <>
      <YourApp />
      {/* Tree-shaken in production */}
      <CiphDevtools />
    </>
  )
}
```

## Features

- Real-time log of all Ciph requests
- Plain vs encrypted body comparison
- Fingerprint cache status
- Zero production overhead

## Production Guard

**Never renders** when `NODE_ENV === "production"` — tree-shaken to 0 bytes.

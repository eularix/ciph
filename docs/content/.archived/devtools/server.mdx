# @ciph/devtools-server

Backend inspector mounted at `/ciph` — view decrypted server logs.

## Setup

```ts
import { ciphDevServer } from "@ciph/devtools-server"

app.route("/ciph", ciphDevServer({
  secret: process.env.CIPH_SECRET!,
  maxLogs: 500,
  password: "dev", // optional
}))
```

## Features

- Live SSE stream of all server logs
- Vanilla HTML/CSS/JS UI (no framework)
- Filter by status/method
- Export JSON
- Password protection

Open `http://localhost:3000/ciph`.

## Production Guard

Returns 404 when `NODE_ENV === "production"`.

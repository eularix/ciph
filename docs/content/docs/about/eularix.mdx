Eularix is a forward-thinking technology company focused on building open-source infrastructure that prioritizes **security without compromise**.

## Our Mission

To create transparent, developer-friendly security tools that protect user data without adding complexity or friction to the development process.

## Core Values

### 🔐 Security First
We believe encryption should be default, not an afterthought. Every design decision prioritizes cryptographic soundness and zero-knowledge architecture.

### 🤝 Open Source
We develop in public, welcome contributions, and believe in community-driven improvement.

### 👨‍💻 Developer Experience
Security tools should feel natural to use. No extensive configuration or sacrificing intuitive APIs.

### 📖 Transparency
Open documentation, clear error messages, and understandable code make our tools accessible to developers of all levels.

## What We Build

### Ciph (Current)
Transparent HTTP encryption for frontend-backend communication. See request/response data only where you need to—DevTools shows ciphertext, devtools panel shows plaintext.

### Planned
- Native language bindings (Go, Rust, Python)
- Protocol-agnostic encryption (gRPC, WebSocket)
- Zero-knowledge authentication layer
- Database-level encryption middleware

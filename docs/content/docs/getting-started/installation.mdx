# Installation

## Client

```bash
pnpm add @ciph/client @ciph/core
```

## Server (Hono)

```bash
pnpm add @ciph/hono @ciph/core
```

## Devtools (Development)

```bash
pnpm add -D @ciph/devtools-client @ciph/devtools-server
```

# Vue (Coming Soon)

Vue.js support for `@ciph/react` is coming in a future release.

For now, you can:
- Use `@ciph/react` directly in Vue (axios-compatible)
- Set up in a composable:

```typescript
// composables/useCiph.ts
import { createClient } from '@ciph/react'

const ciph = createClient({
  baseURL: import.meta.env.VITE_API_URL,
  secret: import.meta.env.VITE_CIPH_SECRET,
})

export function useCiph() {
  return ciph
}
```

```vue
<script setup>
import { ref, onMounted } from 'vue'
import { useCiph } from '@/composables/useCiph'

const ciph = useCiph()
const users = ref([])

onMounted(async () => {
  const res = await ciph.get('/api/users')
  users.value = res.data
})
</script>

<template>
  <ul>
    <li v-for="user in users" :key="user.id">{{ user.name }}</li>
  </ul>
</template>
```

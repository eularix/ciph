# @ciph/core

The foundation of Ciph — isomorphic crypto utilities for browser/Node.js.

## Purpose

Provides AES-256-GCM encryption, HKDF key derivation, and fingerprint generation used by all other packages.

**Not for direct use** — consumed internally by client/server packages.

## Runtime Support

| Runtime | Engine |
|---------|--------|
| Browser | Web Crypto API |
| Node.js | `node:crypto` |
| Bun     | Web Crypto API |

## Key Functions

```ts
// Encryption (used internally)
encrypt(plaintext: string, key: string): EncryptResult

// Key derivation  
deriveKey(secret: string, fingerprint: string): string

// Fingerprint generation
generateFingerprint(components: FingerprintComponents): FingerprintResult
```
